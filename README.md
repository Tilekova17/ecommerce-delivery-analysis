# 📦 E-commerce Delivery Analysis

## 📌 Project Overview

This project analyzes an e-commerce dataset to understand sales trends, customer distribution, and delivery performance.

## 📊 Dataset

Brazilian E-commerce Public Dataset (Olist)

## 🛠 Tools Used

* Python (pandas)
* Google Colab
* Data visualization (matplotlib)

## 🔍 Key Analysis

### 📈 Sales Trend

* Orders increased significantly during 2017
* Peak reached in late 2017 – early 2018
* Drop at the end is due to incomplete data

### 🌍 Customer Distribution

* São Paulo (SP) has the highest number of orders
* Likely due to large population and strong economy

### 🚚 Delivery Analysis

* Average delivery time: ~12 days
* Median delivery time: 10 days
* Some extreme cases up to 200+ days (outliers)

### ⚠️ Key Insights

* Northern regions (RR, AP, AM) have the longest delivery times
* Indicates logistical and geographic challenges
* Potential opportunity to improve supply chain efficiency

## 📌 Conclusion

Delivery performance varies significantly by region. Improving logistics in remote areas could enhance customer satisfaction and reduce delays.
